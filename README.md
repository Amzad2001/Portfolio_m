## Tech Stack

| Layer     | Technology                          |
|-----------|-------------------------------------|
| Framework | React 18                            |
| Build     | Vite 5                              |
| Styling   | Tailwind CSS v4                     |
| Icons     | react-icons, lucide-react           |
| Email     | EmailJS (@emailjs/browser)          |
| Fonts     | Syne · DM Sans · Cormorant Garamond · DM Serif Display |
| Deployment| Vercel                              |

## Sections

- **Hero** — animated name, rotating roles, profile image with ring
- **About** — background and introduction
- **Skills** — tech stack with icons
- **Projects** — featured work with links
- **Services** — what I offer
- **Contact** — working EmailJS contact form

## Running Locally

```bash
# Clone
git clone https://github.com/midhulajphilip/portfolio.git
cd portfolio

# Install dependencies
npm install



# Start dev server
npm run dev
```



## Build & Deploy

```bash
npm run build    # production build → dist/
npm run preview  # preview the build locally
```

Deploys automatically on every push to `main` via Vercel.
